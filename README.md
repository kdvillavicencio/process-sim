# Automating Process Simulations

## Environments
This repository uses `venv` to manage python packages.
### Activating `venv`
```python
source .venv/scripts/activate
```
### Deactivating `venv`
```python
deactivate
```
### Installing Dependencies
```python
pip install -r requirements.txt
```
